Deadline Delirium
=================

[The Linux Game Jam 2019](https://itch.io/jam/linux-game-jam-2019) participant.

![cover](cover.jpg)

Requirements
------------

Nim v0.19.9 (devel)

### Nimble packages installation:

```
nimble update
nimble install sdl2_nim@#head
nimble install nimgame2@#devel
nimble install zip
```
